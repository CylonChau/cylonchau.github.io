interview retrospective 2411

2024-10-31 · 85 字 · 1 分钟  · 
    #Interview

总结：无意义面试，浪费时间

遇到的都是

  1. XXX会不会写，
  2. XXX会不会用

例如，nginx会不会配置，dockerfile 会不会写

比较有效的算，“删除 Pod 处于 termanating，怎么办”。

答：termanating 有多种，不能够优雅处理这个删除，例如Node节点不能通讯，Pod挂在无法卸载，–force就可以，废物面试官说你多看一下。

finalizers：

finalizers in the metadata.finalizers field. When you attempt to delete the resource, the API server handling the delete request notices the values in the finalizers field and does the following:

  • Modifies the object to add a metadata.deletionTimestamp field with the time you started the deletion.
  • Prevents the object from being removed until all items are removed from its metadata.finalizers field
  • Returns a 202 status code (HTTP “Accepted”)

结果总结：

  • 在自我介绍阶段要引导废物面试官的思维，否则他会根据他的认知来影响你，所以自我介绍是非常重要的
  • 面试者和面试官差异向大时，会造成认知度不一致，这种情况下，90%的面试是无效的。
  • 遇到问你会不会xxx，会不会用xxx的直接打断他就可以了，不要在这浪费时间
  • 如果不想去，直接说很高价钱就可以了
